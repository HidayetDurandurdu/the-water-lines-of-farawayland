The Water Lines of Farawayland

A JavaScript grid-based pipe puzzle game where players rotate pipes to connect the water lines.

Features

* Interactive pipe puzzle gameplay
* Rotate pipe pieces to create a connected water path
* Simple HTML, CSS, and JavaScript structure
* Browser-based game with no installation required
* Clean and responsive design

Project Structure

* `index.html` – main webpage structure
* `style.css` – styling and layout
* `script.js` – game logic and interactivity

How to Run

1. Download or clone this repository.
2. Open `index.html` in a web browser.
3. Play the game by rotating the pipes and connecting the water lines.

Notes

* The game runs completely in the browser.
* No database or backend server is required.
